# ElectricityWebAppCalculator

# ⚡ Energy Pro Calc: Ολοκληρωμένος Υπολογιστής Ρεύματος

Το **Energy Pro Calc** είναι μια Web εφαρμογή ανοιχτού κώδικα, κατασκευασμένη με **Python** και **Streamlit**. Σχεδιάστηκε για να προσφέρει απόλυτη διαφάνεια στον υπολογισμό των λογαριασμών ηλεκτρικής ενέργειας στην Ελλάδα.

## 🧐 Τι είναι;
Η εφαρμογή επιτρέπει στον χρήστη να εισάγει τις ενδείξεις του μετρητή του και να λαμβάνει μια **αναλυτικότατη αναφορά** για το πού πηγαίνει και το τελευταίο λεπτό του ευρώ. 

Σε αντίθεση με άλλους απλούς υπολογιστές, το Energy Pro Calc υπολογίζει:
- **Κλιμακωτές ΥΚΩ:** Υπολογίζει αυτόματα τα όρια (kWh) βάσει των ημερών του λογαριασμού.
- **Δημοτικές Χρεώσεις:** Υπολογίζει ΔΤ, ΔΦ και ΤΑΠ με βάση τα τετραγωνικά και την παλαιότητα.
- **Ρυθμιζόμενες Χρεώσεις:** Περιλαμβάνει ΑΔΜΗΕ, ΔΕΔΔΗΕ και ΕΤΜΕΑΡ.

## 🛠️ Τεχνικά Χαρακτηριστικά
- **Πλήρης Παραμετροποίηση:** Όλες οι τιμές (kWh, Πάγια, kVA, Συντελεστές Δήμου) αλλάζουν από το UI.
- **Reactive Interface:** Τα αποτελέσματα ενημερώνονται ζωντανά καθώς αλλάζεις τις ρυθμίσεις.
- **Mobile Friendly:** Σχεδιασμένο για να λειτουργεί άψογα σε κινητά και tablets.
